# Collab Planning Coordination

To set a time to collaborate, we want to establish a time that works for everyone in the collab.

To do so, it's best to be able to suggest a time and let others agree or disagree and suggest a new time.

## User Stories

### 2-person Collab

* Person A creates a new collab room
* Person A sends the room link / code to Person B
* Person B joins the room
* Person A suggests Time X
* Person B sees Time X, but it doesn't work for them so they send Time Y
* Person A accepts Time Y
* Time Y is marked as the new time (both people have accepted)
